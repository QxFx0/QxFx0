# QxFx0 v0.1.0

Release date: 2026-04-29

## Summary

This is the first public baseline of QxFx0 as an alternative, spec-first
Russian dialogue runtime.

## Included

- Typed semantic routing by canonical move families.
- Grammar-oriented Russian rendering path (GF artifacts + runtime mappings).
- Local runtime with CLI and HTTP sidecar interfaces.
- Session-aware worker lifecycle and readiness contracts.
- SQL, Agda, and generated-artifact validation scripts in repository.

## Repository cleanup for public release

- Removed internal audit and research packs from tracked history moving forward.
- Removed intermediate evaluation prompts/reports not needed for runtime use.
- Kept reproducible build/runtime scripts and canonical contracts.

## Scope note

QxFx0 v0.1.0 is a research-runtime baseline, not a general-purpose replacement
for frontier chat systems.
